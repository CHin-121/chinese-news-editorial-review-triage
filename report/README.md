# Course Report Publication Status

The original course report is intentionally excluded from this Phase 1A repository because it contains personal and course-administration information.

A PDF may be added later only as:

```text
course-report-redacted.pdf
```

Before inclusion, the redacted copy must be checked page by page and must remove or safely handle:

- student ID;
- instructor name;
- unnecessary course-submission details;
- private or absolute local paths;
- personal contact information;
- PDF author and other identifying metadata; and
- any raw news article text that should not be redistributed.

Until those checks are complete, no course-report PDF should be published in this repository.
