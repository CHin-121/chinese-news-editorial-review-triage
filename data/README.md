# Dataset

This repository does not include the THUCNews/cnews dataset or any raw news articles.

The original coursework used a cnews subset reported in the course submission to contain approximately 50,000 Chinese news articles across 10 categories. Raw articles are not redistributed through this repository.

Users who later wish to test the archived scripts must obtain an appropriate dataset separately, review its applicable terms, and place local data under:

```text
data/raw/
```

The archived Phase 1A scripts have not been edited and may currently expect a different relative path or filename. Aligning their data paths is reserved for a future reproducibility-cleanup phase.

Phase 1A does not require running the scripts and does not claim successful execution or end-to-end reproducibility.
